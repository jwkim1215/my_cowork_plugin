# text-tools (예시 플러그인)

글쓰기·문서 작업을 돕는 스킬 모음입니다. 마켓플레이스 구조를 보여주기 위한 예시 플러그인이에요 — 자유롭게 수정하거나 새 플러그인으로 교체하세요.

## 포함된 스킬

- **summarize** — 긴 문서나 텍스트를 핵심 요점으로 요약합니다.

## 새 스킬 추가하기

`skills/<스킬이름>/SKILL.md` 형식으로 폴더를 만들면 됩니다. 각 `SKILL.md`는 맨 위에 다음과 같은 머리말(frontmatter)이 필요해요:

```markdown
---
description: 언제 이 스킬을 쓰는지 + 사용자가 말할 법한 문구
---

(여기에 Claude가 따라야 할 지시 내용)
```
